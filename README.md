1. Introducción
MathSnapAI ("nosotros", "la aplicación") es una calculadora científica con inteligencia artificial que respeta tu privacidad. Esta política explica qué datos recopilamos, cómo los usamos y tus derechos.

2. Datos que Recopilamos
2.1 Información de Cuenta
Autenticación anónima: Al usar la app por primera vez, Firebase crea un ID anónimo único para guardar tus créditos.
Google Sign-In (opcional): Si vinculas tu cuenta de Google, almacenamos:
Correo electrónico
Nombre de perfil
Foto de perfil
Estos datos se usan únicamente para sincronizar tus créditos entre dispositivos.
2.2 Datos de Uso
Créditos y compras: Cantidad de créditos disponibles y historial de uso (local y en Firebase).
Historial de cálculos: Expresiones matemáticas resueltas y soluciones de IA (almacenado localmente en tu dispositivo).
Preferencias: Tema visual, modo científico, modo radianes/grados (almacenado localmente).
2.3 Imágenes y OCR
Cámara y galería: Accedemos a la cámara y galería solo cuando tú lo solicitas para escanear problemas matemáticos.
Procesamiento de imágenes: Las fotos se procesan con:
Gemini AI (Google): Reconocimiento y resolución de problemas complejos (requiere enviar la imagen a servidores de Google).
Almacenamiento: Las imágenes NO se guardan permanentemente. Solo se almacenan temporalmente en cache durante el procesamiento y se eliminan al cerrar la app.
2.4 Datos de Terceros (AdMob)
Google AdMob: Mostramos anuncios mediante AdMob, que puede recopilar:
ID de publicidad (Advertising ID)
Dirección IP
Datos de interacción con anuncios
Información del dispositivo (modelo, sistema operativo, idioma)
Para más información: Política de privacidad de Google AdMob
3. Cómo Usamos tus Datos
Funcionalidad de la app: Guardar créditos, preferencias y historial.
Autenticación: Vincular cuenta de Google para sincronización entre dispositivos.
Procesamiento IA: Resolver problemas matemáticos mediante Gemini AI.
Anuncios: Mostrar anuncios relevantes y permitir ganar créditos gratuitos.
Mejoras: Analizar errores y mejorar la experiencia (logs anónimos).
4. Compartir Datos con Terceros
Compartimos datos solo con estos servicios esenciales:

Google Firebase: Autenticación y almacenamiento de créditos.
Google Gemini AI: Procesamiento de imágenes y resolución de problemas (solo cuando usas modo IA).
Google AdMob: Publicidad y anuncios recompensados.
NO vendemos ni compartimos tus datos personales con terceros para marketing.

5. Almacenamiento y Seguridad
Almacenamiento local: La mayoría de tus datos (historial, preferencias) se guardan solo en tu dispositivo.
Cloud (Firebase): Solo los créditos y datos de cuenta se sincronizan con Firebase.
Seguridad: Usamos conexiones HTTPS cifradas y Firebase Security Rules para proteger tu información.
Retención: Los datos se conservan mientras uses la app. Puedes eliminar tu cuenta en cualquier momento.
6. Tus Derechos
Tienes derecho a:

Acceder a tus datos personales
Corregir información incorrecta
Eliminar tu cuenta y todos los datos asociados
Limitar el procesamiento de tus datos
Oponerte al uso de tus datos para publicidad (desactiva personalización de anuncios en ajustes del dispositivo)
Para ejercer estos derechos, contáctanos (ver sección 9).

7. Privacidad de Menores
MathSnapAI NO está dirigida a menores de 13 años. No recopilamos intencionalmente datos de niños. Si descubres que un menor ha proporcionado datos sin consentimiento, contáctanos para eliminarlos.

Notificación en la app (si es un cambio sustancial)
